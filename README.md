# 批量图片编辑 - 在线批量调整尺寸、压缩、加水印、转格式

![批量图片编辑](https://batch-img-edit.pages.dev/favicon.svg)

纯前端批量图片处理工具，所有操作在浏览器本地完成，无需上传服务器，保护隐私安全。

## ✨ 功能特性

- 📤 **批量上传** — 支持拖拽/点击多选图片，一次处理数百张
- 📐 **批量调整尺寸** — 按像素或百分比缩放，可选保持宽高比
- 🔄 **批量格式转换** — JPG / PNG / WebP 互转，支持质量调节
- 📦 **批量压缩** — 调整质量参数压缩图片体积
- 💧 **批量加水印** — 自定义文字水印（字体大小、颜色、位置）
- 👁️ **实时预览** — 处理前后对比，缩略图列表管理
- 📥 **批量下载** — 一键打包 ZIP 下载全部处理结果

## 🛠️ 技术栈

- **纯静态** HTML + Tailwind CSS CDN
- **Canvas API** 处理图片
- **JSZip** 打包下载
- 无 npm、无构建步骤，零依赖部署
